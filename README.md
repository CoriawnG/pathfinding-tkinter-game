# Pathfinding Tkinter Game

Interactive Python Tkinter game featuring **pathfinding enemies, scoring, animations, and collectibles**.  
Built to demonstrate Python, game development, and GUI programming skills.

## Features

- **Dynamic map generation** with walls, coins, start/end points  
- **Player movement** with arrow keys  
- **Multiple pathfinding enemies** with independent speeds  
- **Smooth animations** for enemies  
- **Coin collection** with visual sparkle effect  
- **Player collision flash** on enemy contact  
- **Timer-based challenges**  
- **Score tracking & high score**  
- **Game over and restart functionality** (`R` key)  
- **Pause/Resume button** (works after retry)
